# Credits & Lineage

ImWeb is a browser-based reimagining of Image/ine, the real-time
video synthesis environment designed by Tom Demeyer at STEIM
Amsterdam (1997), developed in close collaboration with Steina Vasulka.

Image/ine was the first software for normal computers that allowed
real-time manipulation of uncompressed video. It set a standard for
what artist-made tools could be — built for video people, not
musicians, not programmers.

ImWeb rebuilds that spirit from scratch for the browser age:
Three.js, WebGL, no installation, no cost, no gatekeeping.

## Dedication
